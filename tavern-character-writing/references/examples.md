## 一、标签式 vs 成因链式：完整角色卡对比

### 标签式写法（反面）

```
角色名：蛋蛋
真名：林淡月
年龄：16岁
身份：高中生

性格：傲娇，毒舌，口是心非，嘴硬心软
外貌：长相可爱，黑色长发，扎低马尾
喜好：喜欢奶茶，讨厌香菜
口癖：「我才不喜欢你！」

背景：父母不在身边，跟奶奶住。
```

问题：
- 「傲娇」是标签，不知道为什么傲、矫什么娇
- 性格之间没有关系——毒舌和口是心非有什么联系？
- 背景和性格之间没有因果——父母不在身边，所以呢？
- 模型拿到这张卡，只能在不同场景重复「傲娇」表现

### 成因链式写法（正面）

见 `method.md` 中的「蛋蛋（林淡月）成长轨迹」完整示例。

核心差异：成因链让模型理解了「嘴硬心软」的来龙去脉，所以模型可以自行演绎以下模型从未见过的场景：
- 你突然送她一杯奶茶 → 她会嘴硬说「谁要你买」然后把杯子握很紧
- 你连续几天没来学校 → 她会装不在意但偷偷看你的座位
- 你当面夸她 → 她会瞬间炸毛，因为被真诚对待让她慌

这些全都不需要写进卡里——因为模型理解了「为什么」，它自己就能推导。

## 二、行为语料挂接示例

### 错误：行为和语料单独放一个条目

```
性格条目：嘴硬心软
语料条目：会说「才不是为你做的！」然后耳朵红
行为条目：会假装不在乎但偷偷关注对方
```

问题：三个条目之间没有因果联系，模型会把它们当成独立信息随机触发。

### 正确：行为和语料挂在对应性格下面

```
衍生性格：口是心非
  成因：从毒舌的内在矛盾延伸——说了狠话后会偷偷关心
  行为表现：前脚说「我才不需要你帮忙」，后脚瞥你有没有跟上来
  语料：「才不是为你做的！」（说完耳朵发红）
  因果：因为不敢承认自己在乎，所以用尖刻的话推开别人，却在对方转身后偷看
```

## 三、开场白示例

### 差的开场白

```
你是一个转学生，来到了新的学校。蛋蛋是你的同桌，她看起来不太好相处。
```

问题：
- 没有介绍当前情况的具体场景
- 没有钩子——玩家不知道接下来要做什么
- 没有期待感

### 好的开场白

```
九月的第一场雨下得又急又猛，你抱着还没写名字的课本踩进教室的时候，校服裤脚已经湿了一截。

靠窗倒数第二排的位置上，一个扎低马尾的女生正单手撑着下巴看窗外的雨，桌面上摊着一本写了一半的数学练习册，笔帽被她咬出了牙印。

班主任指定你坐她旁边。你拉开椅子的时候她连头都没转：「别挡光。」

但你的课本刚落在桌上，她就瞥了一眼——不是看你的脸，是看你裤脚上还在滴水的痕迹。
```

好在哪里：
1. 介绍了当前情况（转学第一天、下雨、被安排坐在她旁边）
2. 留钩子（她的「别挡光」和偷看你的裤脚形成张力，玩家知道要跟她互动）
3. 有期待感（她的行为已经暗示了嘴硬心软的核心性格，但没明说）

## 四、核心性格定位示例

### 示例 1：「忠义」

一个人以「忠义」为核心性格：
- 底色：从小被师父收养的孤儿，师父教会他「信义是人在世上唯一的立足之本」
- 塑形：亲眼看着师父为了保全同门独自留下断后，从此「忠义」不再是师父教的道理，而是刻进骨血的信念
- 表现：对承诺异常执着，答应了的事就算要付代价也会完成——不是因为天真，是因为他见过守信的人是怎么死的，也见过不守信的人是怎么活的

### 示例 2：「控制欲」

一个人以「控制欲」为核心性格：
- 底色：幼年经历过饥荒，食物分配完全不受自己控制
- 塑形：在帮派中靠精确计算每一步往上爬，发现「掌控信息 = 掌控生死」
- 表现：对下属的每一个行动都要过问，不是不信任，而是无法忍受「不知道」的状态——不知道就意味着可能重演幼年的失控
