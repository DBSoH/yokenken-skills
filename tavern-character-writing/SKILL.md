# 酒馆角色卡人设编写

## 概述

用奥尔波特人格特质理论的分层方法手写角色人设：抽象优于具体，解释驱动一切，人类语料是核心资产。

## 核心理念

### 手写 > AI 生成

- 手写卡八股少、人物表现灵活、文风强、文本质量高
- AI 生成的卡文风劣化速度快，即使人工去除了明显 AI 痕迹
- **所有句子都完全是人写的。** AI 只做排版整理，不参与创作本身
- 人类语料非常珍贵——写卡必须保持"人味"，这不是玄学

### 手写并不困难

- 如果你确实有想法，就应该知道自己要什么、不要什么
- 能跟人聊天说出 性癖 和喜欢的角色类型 → 写卡时也完全可以写出来
- 灵感零碎没关系，先想到什么写什么，事后总结整理再补充

---

## 理论基础

### 人格特质三层模型（奥尔波特）

| 层级 | 定义 | 示例 |
|------|------|------|
| **首要特质** | 最典型、最具概括性的核心性格，所有行为均出自它 | 傲娇、忠义 |
| **中心特质** | 日常习惯：怎么说话、怎么做事的主要行为倾向 | 毒舌、温柔、好胜 |
| **次要特质** | 平时不表现，只在特殊场合露出的性格 | 恐高、特定偏爱 |

**关键操作**：由核心性格延伸出衍生性格，衍生性格是对核心性格的进一步阐释。分层结构便于 AI 快速识别重点。

### 解释是一切的核心

**不要把模型当作算命先生**——她猜不到你想要什么，她理解的概念很可能和你想的不一样。

对每一个性格标签进行**解释**，而不仅仅是贴标签：

- ❌ 只写："他是冷静的"
- ✅ 解释："他的冷静体现在：情绪管理能力优秀，情绪比常人更加稳定，不容易生气动怒，而不是像 nerd 一样只会说专有词汇、分析式的学术语言"

**解释的多层含义**：
- 衍生性格 = 对核心性格的解释
- 特定行为和语料 = 对特定性格的具体解释
- 人物经历、家庭环境等 = 都可以作为一种解释

> 从核心到衍生都做了详细解释，甚至补充了人生经历、家庭背景、具体行为表现和语料——这样的角色不可能陷入刻板印象，她会更像一个人。

### 抽象优于具体

**AI 写东西喜欢给具体例子，这直接把人物写死了。** 应该写更抽象的东西——把行为背后的**原因**写出来。

- ❌ AI 方式："她会在 user 帮助她时反而恶语相加，很毒舌"
- ✅ 手写方式："她心里喜欢 user 但是不敢表达，不能坦率地面对自己的情感，所以会言行不一，口是心非"

**原则：先写好抽象部分，再用具体行为做补充。非必要不补充。** AI 能自己演绎的内容不需要写进卡里。只有当角色表现得不合心意、或者你有特定的东西想让 AI 演出时，这才有必要补充。

## 分层结构模板

从抽象到具体的三层嵌套：

```
核心性格
├── 定义: (这个性格是什么)
├── 衍生性格 A
│   ├── 行为表现: (具体做什么)
│   ├── 语料: (具体说什么)
│   └── 衍生性格 A-1
│       ├── 行为表现: ...
│       └── 语料: ...
├── 衍生性格 B (反向性格)
│   └── ...
└── ...
```

- 抽象部分 = 能解释**一类行为**的表现（范围限定）
- 具体部分 = 解读**特定场景**的表现（补充细化）
- 没写抽象只写具体 → 人物很死，限定的范围内表现好，超出去就糟糕

## 写卡流程

### 步骤 1：草稿

**想到什么写什么，大白话即可。**

不需要重视文学修辞——能够清楚地把想法说出来就好。

- 写个人经历、家庭背景、性格成因
- 如果说不清楚：让 AI 帮你整合意图 → 把 AI 写的记住 → **用你自己的话重新写一遍**
- **不要把 AI 整合好的直接拿来用**，那不是人写的，缺少人味

### 步骤 2：提炼整合

用 AI 整理杂乱的思绪，**但所有句子都完全是自己写的**。AI 只基于原句子做格式和顺序调整，内容不变。

提炼提示词：

```
这是我编写的关于人物的解读分析，有些凌乱，但我希望你能全面理解我的意思，并从中梳理出逻辑完善的性格特征。

我希望你能用总分的形式去分层解释人物性格。

比如说一个角色具有一个浓墨重彩的核心性格，所有行为均出自这个性格。比如"忠"关羽。
核心性格，可以延伸出其它性格。比如从关羽的"忠"衍生"义"
核心性格，也可以再延伸出"反向性格"。比如关羽的"傲不容物"
一个人物可能不止一个核心性格。衍生性格、反向性格也同样能做进一步的分层解释。
性格的阐释必须要能做到解释人物一切的行为表现，是抽象的。核心性格是对其他性格的抽象，衍生性格是对核心性格的进一步阐释解读。
衍生性格也可以继续做衍生。
抽象——不仅仅能解读特定场景情境的表现，而是能解释一类的表现。

接下来请你开始梳理我的文本，你不需要增添删减任何原文，也不用画蛇添足自己加新东西。你要做的就是根据我的原始文本，用我的话做人物性格的分层解释。
当然，你需要尽可能多的使用我原文的句子，不需要省略什么东西，且读起来不突兀。我们这里就是越详细越好。同时你可以拆解，打乱重组，进行新的文字编排。就是怎么表达才能最合适，你就怎么来。

文本：
[这里放你写的草稿]
```

提炼完成后，还需微调修改半成品，加入性格层面之外的内容：角色名、社交关系、外貌穿着、居住地等。

### 步骤 3：成品化

将性格部分 + 其余部分交给 AI 换成人设模板格式。

成品提示词：

```
我已经完全编写完成了人物的性格，和一些人设的其他东西。现在需要你将其重新更换一次格式，制作成人设模板。

我整理了一份逻辑完善的性格特征，使用的方法如下：
用总分的形式去分层解释人物性格。一个角色具有一个浓墨重彩的核心性格，所有行为均出自这个性格。
核心性格，可以延伸出其它性格。
核心性格，也可以再延伸出"反向性格"。
一个人物可能不止一个核心性格。衍生性格、反向性格也同样能做进一步的分层解释。
性格的阐释必须要能做到解释人物一切的行为表现，是抽象的。核心性格是对其他性格的抽象，衍生性格是对核心性格的进一步阐释解读。
衍生性格也可以继续做衍生。
抽象——不仅仅能解读特定场景情境的表现，而是能解释一类的表现。

请记住：我的文本，你不需要增添删减任何原文，也不用画蛇添足自己加新东西。你要做的就是根据我的原始文本，为人物更换一份人设模板，因为现在相当于草稿，写的不算好，不是模板。
你需要将我原本文中的内容全部用上，不能省略任何部分，且你用的时候可以做略微的调整，使其读起来并不割裂，不同句子组合起来衔接恰当，没有突兀感。
这也就意味着你可以随意拆解文本，打乱重组，进行全新的文字编排，以最合适恰当的表述呈现。

文本：
[这里放你写好的东西]
```

## 开场白

**开场白是一张卡最重要的部分。** 手写开场白极大提升输出质量。

### 做法

1. 介绍现在什么情况
2. 留钩子让玩家知道接下来怎么走剧情
3. 有期待感，知道大概玩什么、怎么走

### 要点

- 大白话也够用——讲清楚、没有八股就比 AI 强
- 注意规避 AI 爱用的重复句式和词语（八股）
- 可以交给 AI 轻微润色，算半手写开场白，当作文风示例
- **你自己的东西越多，模型能表现出来的就越好**

## 角色补充设定

玩的时候觉得角色表现不对 → 单开一个条目（扮演核心 / 角色指导），用自然语言写。

**原则**：直接说明角色**应该做什么**，不要写"禁止角色 XXXX"——正面指导比禁止更有效。

格式：

```
以下是故事中 XX 角色的一些补充信息，同样很重要：
<角色指导>
用自然语言写你想让人物表现什么，例如"char 不喜欢玩游戏，相比一个人独处，更喜欢和朋友一起出去玩。"
</角色指导>
```

行为表现和语料放在**对应性格下面**，不要单独分开放——语料和行为不是孤立的。

## 快速参考

| 原则 | 做法 |
|------|------|
| **手写** | 所有句子完全由人写，AI 只做排版整理 |
| **分层** | 首要特质 → 中心特质 → 次要特质，从核心到衍生 |
| **解释** | 对每个性格概念进行阐释，告诉模型你真正要什么 |
| **抽象优先** | 先写好"为什么"，再补充"做什么" |
| **非必要不补充** | AI 能自行演绎的内容不写进卡里 |
| **人味** | 人类语料是最珍贵的东西，不要用 AI 翻译你的想法 |
| **开局路径** | 先写自设 → 再写同人 → 最后写原创 |

---

## 常见错误

| 错误 | 正确做法 |
|------|----------|
| 只贴性格标签不给解释 | 每个标签都要阐释"这个性格体现在哪里" |
| 先写具体例子再写抽象原因 | 先写抽象原因，再用具体例子补充 |
| 把 AI 生成的文本直接当内容用 | 用 AI 理解意图 → 自己重新写 → AI 只做整理 |
| 写"禁止角色 XXXX" | 写"角色应该 XXXX" |
| 行为表现和语料单独放一个条目 | 放在对应性格下面，关联起来 |
| 一开始就挑战原创角色 | 自设 → 同人 → 原创，循序渐进 |
| 追求文学修辞不敢下笔 | 大白话写清楚就行，模型理解能力很强 |
