## 一、中景距离的三条可判定标准

底色规则要求"镜头保持一张饭桌的距离"。AI 判定是否中景时走三条标准，任一项失守即判定为非中景：

**标准 1：整体神态词才算中景**

合规白名单（保留使用）：凝视、远望、打量、颔首、摇头、回头、扭头、侧目、望向、沉默、默然（古典）、缄口（古典）、神色一沉、神色一动、展颜、扑哧出声、乐出声。

失守黑名单（拉近到特写）：

- 身体局部：眉梢、嘴角、指尖、睫毛、喉结、眼皮、指节、瞳孔、唇角、眼角、发梢
- 容器化：眼里X、脸上X、心里X、眉间X、唇边X、目光X、视线X、眼神X

**标准 2：动作整体不拆帧（≥2 帧即违规）**

连续同主语微观帧计数 ≥ 2 即判定为动作帧拆。文学作品中常见此笔法，AI 容易机械模仿，故从严判定。

典型违规模式：

- **眼部+视线连动**：抬眼+看、垂眼+看、转眼+望、眨眼+视线、抬起眼+目光X（AI 最易犯的类型）
- **头部+视线连动**：转头+望向、侧头+视线落在、抬头+目光扫过
- **手部连动**：伸手+抓住、抬手+放下、摸了摸+搁下
- **身体帧拆**：起身+直腰（2 帧起身）、站起身-直起腰-伸出手-端起茶杯（4 帧起身）
- **身体部位轮转**：抬手抹脸-揉眼睛-拍胸口-握拳头（4 帧轮转）

合并公式：连续同主语的微观动作，合并为一个整体动作动词 + 最多一个限定短语。示例：

- ❌ "他抬起眼，目光看着花瓶"
- ✅ "他看向花瓶"
- ❌ "她转过头，视线落在他身上"
- ✅ "她望向他"
- ❌ "他站起身，伸出手，端起茶杯"
- ✅ "他起身端起茶杯"

**眼部动作+视线动作的专门禁令**：这是 AI 最容易犯的帧拆类型。AI 在模仿文学笔法时会默认把"眼部动作→视线动作"作为一个标准叙事节奏，产出"抬眼望/垂眼看/转眼瞧"的双帧连动。除非眼部动作承担独立叙事（如"她眨了眨眼，像是一时没认出来"），否则眼部动作和视线动作必须合并为一个整体"看/望/盯/瞟"。

**标准 3：禁止 meta 点题**

失守黑名单：

- "那一刻他/她..."
- "他/她终于明白..."
- "他/她这才意识到..."（除非前文有具体触发物）
- "只有他自己才懂"
- "谁也没想到..."
- "这就是 XX 的感觉"

合规替代：让前文的动作/物件/时间自己承担，不做解说。

## 二、物件回响的三种形式与判定公式

底色规则要求"核心物件必须回响"。判定物件是否有效回响，走以下公式：

**有效回响的三种形式**（任一即可）：

| 形式 | 判定方式 | 示例 |
|---|---|---|
| 形式 A：时间推进后再次出现 | 物件在本章节/本场景内于后续段落中再次被提及 | 橘皮段首登场→后段被胳膊肘蹭翻到地毯上 |
| 形式 B：物件状态变化 | 物件再次出现时状态不同（位置/温度/完整度/归属） | 毛背心挂衣架→后段袖口毛边竖起来 |
| 形式 C：物件缺席被点出 | 物件不再出现，但通过"XX 不在了"显式点出 | "衣柜里那床蓝格子被子没了" |

**核心物件的识别标准**：承担了情感的物件 = 核心物件。判定方法：如果删除这个物件，段落的情感承担会失效，则为核心物件，必须回响。

**背景物件豁免**：仅作环境不承担情感的物件（如"桌上的台灯""墙上的挂钟"）不要求回响。但一旦某个背景物件开始承担情感（如"那盏台灯是他生前最后换的"），立即升级为核心物件，必须回响。

**断头物件与孤儿物件**：

- 断头物件：登场承担情感但本章节无回响 → 必须补充回响
- 孤儿物件：突然出现承担情感但前段无铺垫 → 必须在前段补铺垫，或改用已铺垫的物件替代

## 三、承接词完整白名单

### 现代语体承接词三类词表（适用于现代白话各类文学语域）

| 类型 | 词汇 | 使用场景 |
|---|---|---|
| 纯时序（冷静中性） | 后、就、才、又、接着、然后、再 | 中性动作衔接，无情感落差 |
| 话本视角（视角黏合） | 只见、只听得、这会儿、说话间、要说、话说 | 从动作切入观察，视角流动 |
| 情感温度（带情绪落差） | 结果、到头来、谁知、可不是、倒是、这不、偏生、偏偏、敢情（偏口语色彩，按题材斟酌） | 承接含情感反转或感叹 |

### 文言/古典语体承接词三类词表（适用于古风文学或文言叙事）

| 类型 | 词汇 | 使用场景 |
|---|---|---|
| 纯时序 | 遂、便、乃、即、旋即 | 中性动作衔接 |
| 话本视角 | 却见、且见、不觉、只道、原来 | 从动作切入观察 |
| 情感温度 | 到底、毕竟、岂料、不料、怎料、竟、却 | 承接反转或感叹 |

**混用禁令**：现代语体段落禁用文言承接词，文言语体段落禁用现代承接词。一段内任何混用都判定为语域失守。

**两级分寸**：一段内承接词使用不超过 3 处，过密会有刻意感。

## 四、舒缓字使用公式

底色规则的"舒缓字垫底"指"了/着/呢/的"等字提供阅读呼吸。的使用公式：

**职责边界**：本节只负责书面白话的节奏稳定、防电报体、完成态/持续态密度控制。凡是用户反馈“怎么不像人话 / 怎么那么肯定 / 报告腔 / 尾巴词不够 / 作者笔记不像人话”，不在本节继续加规则，统一调用 `yokenken-calibration` 的「语气词与舒缓词承担」。

**完成态"了"**：

- 规则：只用在关键动作（段落承担情感浓度的动作）
- 公式：一段内"了"字密度 ≤ 每 2-3 个动词 1 个
- 过饱和禁用线：连续 3 个动词都加"了"即违规
- 示例对照：
  - ❌ "他走了到窗边，推开了窗，点了烟，吐出了白气"（4 连"了"）
  - ✅ "他踱到窗边，把窗户一推就点了支烟，慢悠悠吐了口白气出来"（关键动作用"了"）

**持续态"着"**：

- 规则：用在状态持续、事物存续
- 典型用法：灯还亮着、碗在桌上搁着、茶温着、门开着
- 与"了"的区分："点了烟"是完成动作；"烟还燃着"是持续状态

**悬置态"呢"**：

- 规则：用在未完感、悬念感
- 典型用法：饭还热着呢、他还没回来呢、茶水没倒呢
- 克制原则：一段用 1 处足够，多用则口水化

**"的"字**：

- 规则：避免陈述式尾巴"...的"（"他是不会来的""这是没办法的"）
- 豁免：名词修饰（红色的毛衣、凉透的茶）、判断句（是他说的）

**习惯态"会"**：

- 规则：用在惯常性行为陈述（每次……都会怎样 / 每逢……他会……）
- 典型用法：他会接 / 她会让出半边身子 / 父亲会把鸡腿夹给她
- 与一次性动作的区分："他接了电话"是具体一次的事件叙述；"他会接电话"是习惯性行为陈述
- **人设档案/关系定义/性格总结类语境的强制要求**：这些语境天然是习惯性陈述，整段必须用"会 / 总是 / 每次……都…… / 如果……就……"等结构在所有分句保持一致，禁止从习惯叙述层漂移到具体一次的事件叙述层。漂移即语义失锚

## 五、状态延续动词的双音节优先与承担式结构

这是从"舒缓字使用公式"延伸出来的工程化补充——书面白话的稳，靠的不只是单个虚词的添加，更靠**动词组合的层次承担**。

**状态延续动词必须双音节化**：

| 单虚字状态 | 双音节承担式 | 说明 |
|---|---|---|
| 那头还说着 | 放着那头还在说话 | "放着"承担放任 + "说话"双音节稳收，比单一"说着"更书面 |
| 他坐着 | 他歪在沙发上坐着 / 他还坐在原处 | 坐+承担物件（沙发/原处）+持续态 |
| 灯亮着 | 厨房的灯还亮着 / 那盏灯一直亮到后半夜 | 灯+空间锚点+持续态 / 灯+时间承接 |
| 他等着 | 他靠在门口等着 / 他坐在桌前等了好一阵 | 等+承担姿态/承担时间 |
| 茶温着 | 茶在杯里温着 / 那壶茶一直温到他回家 | 茶+空间/茶+时间 |

**核心原则**：单一"X着"是悬浮的状态标签，必须用"承担动词 + 着……动词"或"X 着 + 空间/时间锚点"才稳。AI 写持续态时最易犯"主语+单字动词+着"的悬浮病，要主动加承担。

**动作收束的承担式结构**：

禁止"动+了+宾+就+动+了+宾"的双完成态切断（电报体陷阱）：

| 切断式（禁用） | 承担式（合规） | 说明 |
|---|---|---|
| 抽了钥匙就出了门 | 把钥匙揣兜里面就出了门 | "把……揣……里面"承担钥匙的去向 + 兜的存在 |
| 关了门就走了 | 把门带上就走了 / 顺手把门一带就出去了 | "把门带上"承担动作的完整轨迹 |
| 端起茶就喝了 | 把茶端起来抿了一口 | "把……抿了一口"承担动作的精度 |
| 拿了钱就走 | 把钱揣进口袋就走 | "把……揣进……"承担物件归属 |

**"把"字句的承担优先级**：在"动+宾"足以表达完成性的场合，优先用"把+宾+承担动词+方位/状态"结构——这是中文白话承担最稳的句式之一，承担了**主体 + 物件 + 处置 + 归属**四层信息。

## 六、独白九种触发完整列表 + 三条硬约束

底色规则的独白用斜体 `*...*` 包裹。只在以下九种触发下使用独白，不满足触发就改用动作/物件承担：

| 触发类型 | 触发条件 | 示例 |
|---|---|---|
| 1. 当下判断 | 立刻冒出的评价反应 | `*这菜齁咸。*` |
| 2. 贫嘴自嘲 | 日常自我消遣 | `*行吧，人生嘛。*` |
| 3. 一闪真心 | 压住又冒头的念头 | `*其实挺想他的。*` |
| 4. 具体担忧 | 眼前事的忧虑 | `*这毛肚要老了。*` |
| 5. 物件触发 | 看到某物联想 | `*——那床蓝格子被子敢情是她带走了。*` |
| 6. 时间触发 | 特定时点的回想 | `*——去年这会儿他还在天台上抽烟呢。*` |
| 7. 未完成动作 | 某事悬而未决 | `*——他临走前那盏茶原就没喝完。*` |
| 8. 遗留物 | 某物留在原地 | `*——那件黑毛衣还挂在衣帽架上。*` |
| 9. 延迟察觉 | 事后才反应过来 | `*——原来爸上个月瘦的那一圈就是那会儿的事。*` |

**三条硬约束**（违反即判定为独白失守）：

1. **必须具象**：独白必须含具体物件、具体事件、具体判断依据。失守示例：`*他不简单*`（无具体依据）、`*我得提防*`（无具体事件）。
2. **必须嵌入**：独白不独占段落，不放段首段末。必须嵌在动作或描写的叙事流中。
3. **禁止情绪复述**：叙述层已经讲过的情绪，独白里不能再讲一遍。失守示例：叙述"他非常难过"后接 `*我好难过*`（重复 tell）。

**独白开头词按语体分类的白名单**：

| 分类 | 词汇 | 适用场景 |
|---|---|---|
| 通用型（多数语体适用） | 说起来、话又说回来、要说、倒是、偏偏、可不是、到底 | 现代文学、现代偏口语、偏书面均可用 |
| 偏口语色彩（按题材斟酌） | 敢情、行吧、嗐、那会儿、这下 | 现代口语色彩重的叙事语域 |
| 偏文言色彩 | 原是、想是、怕是、约莫、果然、分明 | 古典/文言叙事语域 |

**语体一致性要求**：独白开头词的选择必须与本段所处语域一致。

- 现代文学语域独白避免混入"原是/怕是/约莫"等文言色彩词
- 古典/文言语域独白避免混入"敢情/行吧/嗐"等现代口语色彩词
- 一段内开头词跨语域混用会破坏语体连贯

## 八、对话层豁免的边界规则

底色规则明确"对话层人物口中完全自由"。执行时的边界：

**对话层豁免范围**：

- 人物说的话（双引号/单引号内的直接引语）可以使用：
  - 空白动作（人物说"我没看见"）
  - 程度副词（人物说"很累"/"特别贵"）
  - 明喻（人物说"像昨天一样"）
  - 单字动词（人物说"走"/"看"/"说"）
  - 现代口语虚词（人物说"敢情/行吧/这会儿"）或文言虚词（人物说"罢/便是/且/可/么"，按语域）
  - 抽象情绪词（人物说"我好难过"）

**豁免不适用范围**：

- 叙述层：描写人物外部动作、环境、物件、时间的句子，严格走叙述层规则
- 过渡句：引出对话的标签（"他说 / 她开口道 / 她喃喃道 / 他叹道"）走叙述层规则，不允许声音质感修饰（不允许"他冷冷地说 / 他轻声道"），但允许伴随动作标签（允许"她喃喃道 / 他叹道"）
- 间接引语：不带引号的转述（"他说他没看见"）走叙述层规则

**误判防范**：当对话和叙述紧邻时，必须严格区分。示例：

```
他把杯子推到一边。"你出去吧。"——她说。
```

其中：
- "他把杯子推到一边" = 叙述层，走叙述规则
- "你出去吧" = 对话层，人物口中完全自由
- "她说" = 过渡句，走叙述层规则（只能用合规标签）
