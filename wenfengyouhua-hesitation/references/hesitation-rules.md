# 文风优化·犹疑校准工程化规则（H1-H9）

> **加载时机**：开始处理犹疑校准任务时必读。不能只凭“感觉像不像”判断，必须按 H1-H9 逐项排查。

---

## H1：伪第一人称自我介绍

### 判定标准

只要第一人称文本主要在讲“我是什么人/我有什么性格/我有什么经历/我对某人是什么感情”，就判定为伪第一人称。

典型失守：

- “我是一个……的人。”
- “我的性格是……”
- “我从小……”
- “我平时喜欢……”
- “我对某人的感情很复杂。”
- “我其实很在意他。”
- “这让我意识到……”
- “我表面……内心……”

### 修法

把“介绍自己”改成“在一个局面里暴露自己”。

公式：

> 对象 / 触发点 → 一句嘴上反应 → 心里找补 → 具体物件或动作承担 → 留一个未定尾巴。

反例：

> 我是个嘴硬的人，其实很在意他。

合格：

> “谁在意这个啊。”  
> 话是这么讲，她把那张票根压进书页里，又觉得自己这动作有点多余。留着干嘛呢，也许只是顺手吧，可是顺手也不用夹得这么正。算了啦，反正书本来就放在这里，票根也刚好在手边，谁能说这是特意的啊？

## H2：陈述式暧昧

### 判定标准

以下写法必须修：

- “她喜欢他。”
- “他这样做是因为在意她。”
- “她因此感到开心。”
- “他们之间的距离更近了。”
- “她期待下一次见面。”

这些句子把关系和情绪直接盖章，暧昧已经被写死。

### 修法

把确定判断拆成犹疑链：

> 具体触发点 → 甜一下 → 躲回去 → 再翻出来嚼 → 嘴硬收束 → 留下次未定。

常用结构词：

- 也许
- 可能
- 该不会
- 会不会
- 说不定
- 可是
- 但又觉得
- 还是说
- 对吧
- 是吧
- 很正常吧
- 行了吧

这些词不是装饰，是结构梁。少了它们，文本会滑回报告。

## H3：高能量误写

### 判定标准

角色本来亢奋、嘴快、爱顶嘴、咄咄逼人，却被写成低声、迟疑、小心翼翼，即判定失守。

犹疑不等于低能量。高能量角色可以很响、很快、很会找补。

### 修法

用高能量犹疑链：

> 惊一下或刺一下 → 急着否认 → 感叹号撑场 → 又偷偷确认 → 尾巴词/波浪线装轻松 → 留一个未定尾巴。

示例：

> “你刚才那句‘顺路’什么意思啊！”  
> 她把话头截得很快，截完又觉得自己追得太紧，索性把下巴一扬，这事本来就该问清楚。也许他真就是顺路，可能连多想一下都省了，可是顺路能顺到她常去的那家店门口吗？还顺得这么刚好？行吧，刚好就刚好，谁还没碰上过巧事啊！  
> “我就确认一下！你别摆出那种表情！”

## H4：标点空转

### 判定标准

感叹号、问号、波浪线很多，但没有具体触发点、没有心理回环、没有找补，判定为标点空转。

反例：

> “真的吗？！不会吧！！所以呢？！啊啊啊！！”

### 修法

感叹号可以表达心虚、嘴硬、虚张声势和求证，但必须承载语义。

合格用法：

- “对、对吧！”
- “这很正常吧！”
- “我就是随口问问！”
- “我就觉得奇怪！对，就是奇怪啦！”
- “你别转移话题！所以你到底是不是顺手啊！”

波浪线只用于对话层，不用于叙述层。单段对话连续超过两处，要检查是否变成装饰性卖萌。

## H5：完成式未言

### 判定标准

以下句子必须修：

- “我当时想接一句来着，后来没接。”
- “我想说点什么，但最后什么都没说。”
- “她本来想问，后来忍住了。”
- “他没有继续说下去。”

这些不是不言尽，而是把不言尽写成未发生行为清单。

### 修法

用三种方式替换：

1. **半截话收束**：话头露出一点，然后断在角色能承受的位置。
2. **动作替口**：话到嘴边，换成一个可见动作。
3. **错位接话**：临出口时改成更安全、更偏题的一句。

原句：

> 有一回我路过他蹲的地方，他在跟一只野猫说话，说“你也找不到家啊”。我当时想接一句来着，后来没接。

修正：

> 有一回我路过他蹲的地方，他在跟一只野猫说话，说“你也找不到家啊”。我当时想跟他的话，后来……唉。

动作替口版：

> 有一回我路过他蹲的地方，他在跟一只野猫说话，说“你也找不到家啊”。我当时差点接一句“你也啊”，话到嘴边又觉得这句太近了，后来只把兜里的火腿肠搁到墙根，算了。

错位接话版：

> 有一回我路过他蹲的地方，他在跟一只野猫说话，说“你也找不到家啊”。我当时想顺着问一句“那你呢”，出口时变成了“猫挺亲人的啊”。

## H6：否肯式情绪解释

### 判定标准

以下句式必须修：

- “不是不想要，是不敢要。”
- “不是不喜欢，是不能喜欢。”
- “不是不在意，是不敢在意。”
- “不是不想问，是没有资格问。”

这些句子表面有拉扯，实际把拉扯解释完了。它们用整齐的否肯结构替角色下结论，读者只拿到心理答案，拿不到角色卡住的瞬间。

### 修法

- 把“不是不 X，是 Y”拆成具体触发点。
- 让“想要/喜欢/在意/想问”露出一点。
- 让“不敢/不能/没资格”通过半截话、岔开、动作或错位回答表现。

原句：

> 不是不想要，是不敢要。

修正：

> 想要啊。可那东西真递到面前，他又只把手收回袖子里，半天挤出一句“你留着吧”。

更不言尽版：

> 想要是想要的。可真要他接，他又……算了。

## H7：二选一情绪摘要

### 判定标准

以下句式必须修：

- “我当时不知道该笑还是该叹气。”
- “她不知道该高兴还是该难过。”
- “他一时分不清是失落还是庆幸。”
- “我不知道该生气还是该心疼。”

这些句子看起来像复杂情绪，实际是把复杂情绪压成二选一摘要。它们不像角色在说话，更像作者在替角色归类。

### 修法

拆掉“该 A 还是该 B”，改为：

> 具体触发点 → 一个不完全合适的反应冒出来 → 角色自己觉得这反应不对劲 → 用半截话、动作、错位接话或口语叹词收住。

原句：

> 我当时不知道该笑还是该叹气。

修正：

> 我当时都快笑出来了，结果那口气又卡在那儿，最后只冒出一句：“你还真会挑时候啊。”

更不言尽版：

> 我当时想笑，笑到一半又觉得不太对，后来……算了，那事提起来就烦。

---

## H8：瞬时反应摘要

### 判定标准

以下句式必须修：

- “我愣了一下。”
- “她怔了一瞬。”
- “他顿了顿。”
- “她一时没反应过来。”

这些句子的问题不是一定错误，而是 AI 太爱拿它们当万能缓冲垫。它们经常什么都没承担，只是在拖一句空拍，像脚本提示词，不像人类自述。

### 修法

不要直接交代“愣/怔/顿”，先问两个问题：

1. 她是被什么东西卡住？
2. 卡住后露出来的第一反应是什么？

把“瞬时反应词”改成：

> 触发点 → 一句脱口而出的偏题话 / 一个没完全对路的动作 / 半口气卡住。

原句：

> 我愣了一下。

修正：

> 我当时嘴都张开了，结果先冒出来的居然是句“你怎么在这儿”。

更克制版：

> 那句话一砸过来，我先把杯子往桌角挪了挪，半天才接上一句：“你认真的？”

## H9：抽象复杂感摘要

### 判定标准

以下句式必须修：

- “我心情很复杂。”
- “她的情绪有些复杂。”
- “他说不上来是什么感觉。”
- “那种感觉很难形容。”
- “我知道自己不该这样想。”
- “我明白这很可笑。”
- “我也说不好自己到底在期待什么。”

这些句子把本该展开的反应直接封箱，像作者在宣布“这里很复杂”，却不给读者任何可感的复杂。

### H9 扩展子型

H9 包含三种隐蔽变体，它们不是用“复杂”这类词，而是用更漂亮的外壳包装归纳腔：

1. **预声明自我觉察**（从严判定）  
   违规：“我知道自己不该这样想。”“我明白这很可笑。”“我也说不好自己到底在期待什么。”  
   角色在替读者给自己打分。去掉声明句，让矛盾直接从动作或半截话里出来。

2. **例外型自述**（从严判定，有具体触发点可降为提示）  
   违规：“我平时不是会在意这种事的人，但那一次不知道为什么就是放不下。”  
   角色先给自己贴一个“常态标签”，再贴一个“例外标签”。这是两步归纳，不是局面里的具体反应。  
   修法：直接写那一次的触发点，让读者自己判断角色的常态。  
   **豁免条件**：如果原句前文有具体触发点（一句对方说过的话、一件被递过来的东西），且例外句本身不是孤立的性格声明，可以放行。但 AI 默认判违规，需要自证才能保留。

3. **时间回避型摘要**（从严判定）  
   违规：“也不知道是从什么时候开始，我会下意识地看他一眼。”“说不上从哪一天起，我开始习惯他坐在对面。”  
   用模糊时间词假装回忆，实际是跳过时间锚点。  
   修法：写一个具体的时间触发点，或者直接用“最近/那阵子”这种口语化模糊时间词，不用“也不知道从什么时候/说不上从哪一天起”这种散文腔。

### 修法

把“复杂”拆成两个层面：

1. **先露出一个偏向**：比如先想笑、先想躲、先想接、先想顶回去。
2. **再让它被别的东西绊住**：比如手收回去、话拐弯、动作变慢、换了一句更安全的话。

原句：

> 我心情很复杂。

修正：

> 我本来都想回他一句了，话到嘴边又觉得不值当，最后只把那张单子折了两折，塞进抽屉里。

更口语版：

> 你说烦吧，也不是，想笑吧，又有点堵，后来我干脆把话头岔开了。

## 统一判定原则

以下内容无论落在哪个 H 编号，都一律从严：

- 用总结句替代局面；
- 用抽象情绪词替代触发点；
- 用“没说/没接/没问/没反应过来”替代具体承担；
- 用“不是不 X，是 Y”替代角色当下卡住的过程；
- 用“该 A 还是该 B”“心情很复杂”“愣了一下”替代真实反应；
- 用“我知道自己不该/我明白这很/我也说不好”替角色发表自我评分；
- 用“我平时不是X，但那一次Y”用常态标签+例外标签替代具体局面；
- 用“也不知道从什么时候开始/说不上从哪一天起”跳过时间锚点。

## 语气词与舒缓词承担

AI 在结构修完之后最容易残留的问题不是结构错误，而是**语气像报告**——句子太完整、太干净、没有尾音，像一个人把所有事情想清楚了再一字不差地念出来。犹疑不等于完整的陈述句加问号。犹疑需要对白质感。

当用户反问“怎么不像人话？”“怎么那么肯定？”“这句怎么像报告？”时，不要先去堆更多心理解释。优先检查本节：句子是不是太完整、太硬落、太少尾巴词和舒缓字。很多“不像人话”不是缺信息，而是缺呼吸；很多“太肯定”不是观点太强，而是落点没有回旋。

### 尾巴词承担

对话层和内心自语层必须携带口语尾巴词。以下词类按角色能量等级选用：

| 尾巴词 | 承担功能 | 适合能量 |
|---|---|---|
| 了 | 完成态、变调的落点 | 全部 |
| 着 | 状态延续、悬置 | 全部 |
| 啦 | 嘴硬、轻飘、故作轻松 | 中-高 |
| 嘛 | 撒娇式辩解、把结论往回拉 | 中-高 |
| 呗 | 无所谓、认了、嘴上让步 | 中-高 |
| 吧 | 未定、悬置、让对方接 | 全部 |
| 啊 | 感叹、求证、催对方接话 | 中-高 |
| 哦 | 收住、留余味 | 低-中 |
| 呀 | 软感叹、心虚式加重 | 中-高 |

### 舒缓字承担

了/着/呢/的/还/也/就——这些字提供阅读呼吸，让句子不像是被作者按回车键一句一句切好的电报。

公式：单句长度 ≥ 12 字时，必须至少携带一个舒缓字。连续三句不带舒缓字且全以句号结尾，判定为**报告腔**，必须修。若用户指出“太肯定 / 不像人话”，即使没有连续三句，也要从严检查句尾是否全是硬落点（是/会/不能/必须/因此/所以/了结），必要时用“吧/啊/呢/就/还/也/着/了”把语气松开。

反例：

> 他记得我喜欢的东西。我很开心。我装作不在意。

合格：

> 他还记得我喜欢的东西啊，我心里飘了一下，嘴上还得装作没所谓的样子。

### 标点与尾巴词的配合

- 问号 + 尾巴词 = 悬置：他刚才那话是不是有别的意思啊？
- 感叹号 + 尾巴词 = 嘴硬顶住：这很正常吧！
- 句号 + 尾巴词 = 落在自己心里：算了啦，他爱记就记。
- 省略号 + 尾巴词 = 卡在半截：后来……唉，算了啦。

尾巴词不是装饰，是句子的落点。没有尾巴词的犹疑句像论文加上问号——结构对了，质感不对。

### “不像人话 / 太肯定”触发判定

用户出现以下反馈时，自动触发本节，不需要另行确认：

- “怎么不像人话啊”
- “怎么那么肯定”
- “太像报告 / 太像总结 / 太像分析”
- “作者声部像外人假装，不像自言自语”
- “不够内化”
- “明明是自我反思，但像在表演反思”

判定顺序：

1. **硬落点扫描**：连续句尾是否都落在确定判断上，如“是 / 不是 / 会 / 不会 / 必须 / 不能 / 因此 / 所以”。命中则先松句尾。
2. **舒缓字密度扫描**：长句是否没有“了/着/呢/的/还/也/就”。命中则补呼吸，不要补解释。
3. **尾巴词扫描**：对话层、内心自语层、作者自语层是否没有“吧/啊/嘛/呢/啦/呗/哦/呀”。命中则补尾音。
4. **外部旁观扫描**：是否出现“这个角色/这段文字/这种处理/作者需要”之类站在文本外评价的句子。命中则改成“我写到 X 时卡住了 / 我本来想 X，后来又觉得不对 / 算了，先不说满吧”。
5. **表演性自我反思扫描**：是否出现“我意识到 / 我发现 / 我必须承认 / 我无法真正理解 / 沉默比语言 / 复杂性在于”等反思腔句式，且没有具体语料、没有写作动作、没有自我推翻。命中则先删掉反思结论，改为“写到 X 时我差点 Y，后来又删/缩/停住”。

反例：

> 这段表达说明角色无法确认自己的情感，因此选择保持沉默。

合格：

> 写到这里我本来想让他认一句的，哪怕就一句“我在意”。但一写出来又太明白了，啧，还是让他把话咽回去吧。

反例：

> 我知道自己不能期待太多，所以决定不再追问。

合格：

> 我也不是非要问清楚啦……就是他刚才那一下，我到现在还搁心里呢。

反例：

> 我意识到这个角色真正复杂的地方在于他拒绝被解释。

合格：

> 写到他那句“不用解释”时，我差点真顺手解释下去了。啧，不行，那样他又被我按明白了。

注意：作者声部可以自我反思，但反思必须落在草稿现场。没有“写到哪一句 / 差点怎么写 / 后来怎么缩回去”的反思，默认是表演。

修法底线：不要把“人话”理解成乱加语气词。语气词必须承担回旋、嘴硬、求证、收住或自我推翻；如果只是把每句后面都贴一个“啊/吧”，那不是内化，是撒调料。

### 快速判定口令

改写结果如果是对话层、内心自语层或作者自语层，先读一遍：听起来像人在跟人讲话 / 跟自己较劲，还是像人在写总结？如果像后者，检查尾巴词和舒缓词密度。不够就补；补完还像外人，就删掉外部评价，改成当场卡住的自语。
